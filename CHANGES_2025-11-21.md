### 2. Çekirdek Dağılım Analizi Modülü

**Yeni Dosya**: `pfaz_modules/pfaz01_dataset_generation/nuclei_distribution_analyzer.py`

**Özellikler**:
- **Z Dağılımı**: Proton sayısı dağılımı (min, max, mean, median, unique count)
- **N Dağılımı**: Nötron sayısı dağılımı
- **A Dağılımı**: Kütle sayısı dağılımı
- **Kütle Grupları**: Hafif, orta, ağır çekirdekler
- **Çekirdek Tipleri**: even-even, odd-odd, even-odd, odd-even
- **İzotop Çeşitliliği**: Her element için izotop sayısı
- **İzoton Çeşitliliği**: Her N değeri için farklı Z sayısı
- **Magik Sayı Analizi**:
  - Magik Z ve N (2, 8, 20, 28, 50, 82, 126, vb.)
  - Magike yakın çekirdekler (±2)
  - Çift magik çekirdekler (örn: He-4, O-16, Pb-208)
- **Deformasyon Analizi**: Beta_2 istatistikleri (min, max, mean, std, bölgeler)

**Metodlar**:
```python
class NucleiDistributionAnalyzer:
    def analyze_dataset(df, dataset_name) -> Dict
    def create_distribution_report(analysis, save_path)
    def create_master_nuclei_catalog(df_full, output_path)
```

## 📊 Dağılım Raporu İçeriği

`nuclei_distribution_report.xlsx` dosyası şu sheet'leri içerir:

1. **Summary**: Genel istatistikler
   - Toplam çekirdek sayısı
   - Z, N, A aralıkları
   - Unique değerler
   - Çift magik çekirdek sayısı

2. **Z_Distribution**: Proton dağılımı
   - Her Z değeri için count

3. **N_Distribution**: Nötron dağılımı
   - Her N değeri için count

4. **A_Distribution**: Kütle dağılımı
   - Her A değeri için count

5. **Mass_Groups**: Kütle grupları
   - Light (A<40)
   - Medium-Light (40≤A<90)
   - Medium (90≤A<140)
   - Medium-Heavy (140≤A<200)
   - Heavy (A≥200)

6. **Nucleus_Types**: Çekirdek tipleri
   - even-even (count & %)
   - odd-odd (count & %)
   - even-odd (count & %)
   - odd-even (count & %)

7. **Isotope_Diversity**: İzotop çeşitliliği
   - Her element (Z) için izotop sayısı
   - Ortalama izotop sayısı
   - Maksimum izotopa sahip elementler

8. **Magic_Numbers**: Magik sayı analizi
   - Exact Magic Z sayısı
   - Exact Magic N sayısı
   - Near Magic Z sayısı (±2)
   - Near Magic N sayısı (±2)
   - Çift magik çekirdekler

9. **Deformation_Stats**: (Beta_2 target için)
   - Min, max, mean, std

10. **Deformation_Regions**: (Beta_2 target için)
    - Spherical
    - Weakly deformed
    - Moderately deformed
    - Highly deformed

## 📝 MATLAB Entegrasyonu

### .mat Dosya İçeriği

**train.mat**:
```matlab
train_input      % Features matrix (n_samples × n_features)
train_output     % Target values (n_samples × n_targets)
feature_names    % Cell array of feature names
target_names     % Cell array of target names
```

**check.mat & test.mat**:
```matlab
check_input / test_input    % Features
check_output / test_output  % Targets
```

### Kullanım Örneği

```matlab
% Veriyi yükle
data = load('train.mat');
X = data.train_input;
y = data.train_output;

% ANFIS eğit
fis = anfis([X y]);

% Test et
test = load('test.mat');
pred = evalfis(fis, test.test_input);
```
