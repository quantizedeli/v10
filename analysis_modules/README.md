# Analysis Modules

## Description

Analysis modules for model interpretability, robustness validation, and real data integration.

## Modules

- `model_interpretability.py` - Model interpretability and explainability
- `robustness_validation_manager.py` - Robustness validation framework
- `real_data_integration_manager.py` - Real experimental data integration

## Usage

```python
from analysis_modules import model_interpretability

# Analyze feature importance
importance = model_interpretability.analyze_feature_importance(model, X)
```
