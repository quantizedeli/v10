# PFAZ 06: Final Reporting

## Description

Final Reporting - Comprehensive Excel and LaTeX report generation with charts and formatting.

## Modules

- `pfaz6_final_reporting.py` - Main reporting orchestrator
- `comprehensive_excel_reporter.py` - Excel report generation
- `excel_formatter.py` - Excel formatting utilities
- `excel_charts.py` - Chart generation for Excel
- `latex_generator.py` - LaTeX report generation
- `advanced_analysis_reporting_manager.py` - Advanced analysis reports
- `reports_comprehensive_module.py` - Comprehensive reporting system

## Usage

```python
from pfaz_modules.pfaz06_final_reporting import pfaz6_final_reporting

# Generate reports
pfaz6_final_reporting.generate_all_reports(results)
```
